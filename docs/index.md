# Cursor AI Guide

Работа с Cursor как с инженером. Практики из telnik-flight-engine, Airblox, Telnik Platform.

## Кому

- Новичкам: Cursor не будет выдумывать код
- Опытным: ускорение работы без разрушения архитектуры

## Что получите

- Настройте контекст — Cursor перестанет фантазировать
- Даёте задачи по шагам — получаете предсказуемый код
- Шаблоны промптов: роль → контекст → задача → критерии → формат
