# Cursor AI Guide

Как работать с Cursor как с инженером: контекст, промпты, дизайн, реализация. Основано на практике в telnik-flight-engine, Airblox и Telnik Platform.

## Для кого
- Новичкам — чтобы Cursor не выдавал «магический» код.
- Опытным — чтобы ускорять работу и не ломать архитектуру.

## Что получите
- Настроите контекст так, чтобы Cursor не выдумывал.
- Ставите задачи по шагам — получаете предсказуемый код.
- Готовые шаблоны: роль → контекст → задача → критерии → формат ответа.
