# Cursor AI Guide

Практики работы с Cursor как инженером: промпты, контекст, архитектура.

Основано на опыте работы в telnik-flight-engine, Airblox и Telnik Platform.

## Для кого

- Новичкам: Cursor не будет выдумывать код
- Опытным: ускорение работы без разрушения архитектуры

## Что внутри

- Принципы работы с Cursor
- Структура эффективного промпта
- Проектирование и реализация с Cursor
- Техники общения и готовые workflows
- Типичные ошибки и как их избежать
- Примеры из реальных проектов

## Документация

Полная документация доступна на [GitHub Pages](https://it1ro.github.io/cursor-guide).

## Локальный запуск

```bash
pip install -r requirements.txt
mkdocs serve
```
